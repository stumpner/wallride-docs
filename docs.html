Intro

What is WallRide?

WallRide is a multilingual easy-to-customize open source CMS made by Java, using Spring Framework, Hibernate and Thymeleaf.
WallRide focuses on sophisticated UI, simple and clean source code and easy deploy to AWS BeansTalk(of course other servlet containers)
We hope this CMS is loved by many developers of principles all over the world.

Getting started

WallRide has 2 packaging types; jar and war.

Installation requirements

  • Java8 or higher
  • Database (MySQL or PostgreSQL)

Executable jar

What you need to do:

1.Download the latest jar files from the link below. [link] 2. Create wallride.home
3. Create application.properties
4. Execute command
5. Access URL below
5. See setup

War + Tomcat

What you need to do:

1.Download the latest war file from the link below.
[link]
2. Create wallride.home
3. Create application.properties
4. Configure context in Tomcat
5. Modify Tomcat's server.xml to handle internationalized characters correctly
6. Fix memory and mail handling settings in Tomcat
7. Start Tomcat
8. See setup

AWS Elastic Beanstalk

1.Download the latest war file from the link below.
[link]
2. Crerate wallride.home on S3
3. Crerate application.properties on S3
2. Setup AWS Elastic Beanstalk
3. Configure AWS Elastic Beanstalk
4. Upload war
